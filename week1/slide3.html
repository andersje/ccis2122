1. power on
  2. firmware (e.g. BIOS) Check
  3. firmware loads bootloader
  4. bootloader (eventually) loads kernel
  5. kernel mounts /boot RO
  6. kernel grabs data from /etc/fstab, checks filesystems
  7. kernel remounts filesystems RW
  8. kernel forks, starts init
  9. init follows rules in /etc/inittab
