• Here's our basic steps:
    1. make a dir tree of what you want to package
    2. Run this command, from the parent of that directory:
      fpm -s dir -t rpm --name myLousyPackage

      Your package will be in myLousyPackage-1.0-1.x86_64.rpm
    3. consider adding pre/post config scripts
    4. Check it all into git
