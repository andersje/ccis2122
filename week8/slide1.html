• SMB vs. NFS
  • Developed by IBM, Intel & MS ~1985-7
  • NFS was released in 1985
  • connections via NetBIOS over TCP/IP, IPX/SPX, or NetBEUI
  • SMB protocol is named for the underlying SMB (Server Message Block) message structure
  • SMB protocol docs are at ftp://ftp.microsoft.com/developr/drg/CIFS/
  • SMB is now known as CIFS: Common Internet File System
