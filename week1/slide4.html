• SVR4 has runlevels, BSD does not
  • Redhat/CentOS/Fedora/Mandriva runlevels:
    • 0: system halted
    • 1: single-user/maintenance mode
    • 2: multi-user, NFS deactivated, text login
    • 3: multi-user, NFS activated, text login
    • 4: unused/site-defined
    • 5: multi-user, graphical (e.g. xdm) login
    • 6: reboot
  • different distributions may be slightly different: SuSE
  • runlevel command tells you what runlevel you're at.
  • init X will instruct init to take the system to runlevel X
