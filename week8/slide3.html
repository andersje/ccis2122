• samba config file is in /etc/samba/smb.conf
  • samba passwords are in /var/lib/samba/private/*
  • unix users are mapped to windows usernames in /etc/samba/smbusers
  • smbpasswd -a freyja adds the user named freyja
  • /etc/rc.d/init.d/smb restart restarts the server
