• Xen is currently the 800-pound gorilla of linux hardware virtualization
  • does paravirtualization through a modified kernel
  • given Socket AM2 processors, or Intel C2Ds, full virtualization is an option
  • heavily integrated into Redhat/CentOS since Fedora Core 6
  • install via Applications -> Add/Remove -> Base System -> Virtualization
  • this gives us xen + some management tools
