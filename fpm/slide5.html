• Now, we want to run commands after install. How?
  • Here's a script:
    #!/bin/bash
echo 'echo "unix down, please reboot" ' >> /etc/motd
chmod a+rx /usr/local/share/doc/NameOfOurPackage
chmod a+r /usr/local/share/doc/NameOfOurPackage/README.md
  • fpm -s dir -t rpm --name mypkg -v 1.1 --after-install ./ExampleScript.bash ./mypkg
  • Want to make a deb? The command changes to this:
    fpm -s dir -t deb --name mypkg -v 1.1 --after-install ./ExampleScript.bash ./mypkg
  • Warning: puppet/chef is probably better for config logic!
